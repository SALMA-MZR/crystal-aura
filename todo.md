# Landing Page Skincare - Plan de Développement

## Directives de Design

### Références de Design
- **Style**: Modern Minimalism + Natural Beauty + Soft Elegance
- **Inspiration**: Premium skincare brands (La Roche-Posay, CeraVe, The Ordinary)

### Palette de Couleurs
- Primary: #2D5F4F (Vert naturel profond)
- Secondary: #E8F4F0 (Vert pastel clair)
- Accent: #D4A574 (Or rose - CTAs)
- Background: #FFFFFF (Blanc pur)
- Text: #2C3E50 (Gris foncé), #6C757D (Gris moyen)

### Typographie
- Headings: Playfair Display (élégant, serif)
- Body: Inter (moderne, sans-serif)
- H1: 48px, font-weight 700
- H2: 36px, font-weight 600
- H3: 24px, font-weight 600
- Body: 16px, font-weight 400

### Composants Clés
- **Boutons CTA**: Background #D4A574, texte blanc, border-radius 30px, hover: darken 10%
- **Cards**: Background blanc, border 1px #E8F4F0, border-radius 15px, shadow subtile
- **Icons**: Font Awesome 6, couleur #2D5F4F

### Layout & Espacement
- Hero: Full viewport height avec image de fond
- Sections: padding 80px vertical
- Container: max-width 1200px
- Grid produits: 3 colonnes desktop, 2 tablette, 1 mobile
